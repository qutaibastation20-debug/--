منصة دروس أونلاين مع الأستاذة إسراء توفّر طريقة سهلة وآمنة لحضور الحصص المباشرة لطلاب الصف الأول إلى السادس. يدخل الطالب بكود خاص به ليصل مباشرة إلى رابط الدرس عبر Zoom، بينما تستطيع المعلمة إدارة الأكواد وروابط الحصص من لوحة تحكم بسيطة. تصميم جذّاب وتجربة استخدام مريحة لتنظيم الدروس الأونلاين بكفاءة.
